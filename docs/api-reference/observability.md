# `ciel.observability` — Auditoría y trazas

Trazabilidad multi-tenant para Ciel: eventos de auditoría (`AuditEvent`),
sumideros (`AuditSink`) y un trazador consciente de herramientas
(`ToolAwareTracer`) que emite spans a través de un sumidero asíncrono.
